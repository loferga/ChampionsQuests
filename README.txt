command list:
newtab <tabName>                   // add a new tab and initialize entry values
add <entryName>                    // add an entry with all values initialized
purchase <entryName>               // initialize all values to default unchecked
refund <entryName>                 // null all values except the checked ones
check <entryName> <tabInteger>     // check a value on a tab
uncheck <entryName> <tabInteger>   // uncheck a value on a tab
remove <entryName>                 // completely remove an entry
deletetab <tabInteger>             // delete a tab and enties vallus of this one
next <tabInteger>                  // print the next entry to check on a tab
